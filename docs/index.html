Add to Reminders — one-time setup

Add to Reminders hands checked items to a Shortcut, which drops each one into your Groceries list as its own reminder. Takes about a minute, once:

  1. Open the Shortcuts app → tap + for a new shortcut.
  2. Add action Split Text → input Shortcut Input, separator New Lines.
  3. Add action Repeat with Each → set it to repeat over the Split Text result.
  4. Inside the repeat, add Add New Reminder → reminder: Repeat Item → list: Groceries. (The loop is what makes each line its own reminder — pointing Add New Reminder at the whole Split Text list instead just creates one lumped-together reminder.)
  5. Tap the shortcut's settings (⋯), rename it exactly "Add to Groceries", then Done.

Don't want to set that up? Tap Share… instead — same list, no setup, but Reminders will file it as one reminder with everything in its notes rather than separate checkable items.
